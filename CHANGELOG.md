# MagicStage 更新日志

## 2.0.1（Build 9）— 2026-07-19

### 修复

- 修复 2.0.0 更新后应用无法启动（系统提示"已损坏"）的问题：改用 Xcode 自动签名，CodeDirectory v=20500、Executable Segment flags=0x1、完整 entitlements（含 `get-task-allow` 与 `files.user-selected.read-only`）。
- 修复 Release 编译时 Swift 6.3 EarlyPerfInliner 优化 `deinit` 崩溃（`TransparentHostingView`、`NativeSettingsPopUpButton.Coordinator`）。

### 发布说明

- 2.0.0 用户因旧版本无法启动，**不能通过 Sparkle 自动更新**，请手动下载 2.0.1 覆盖安装。
- 最低支持 macOS 14.0。

## 2.0.0（Build 8）— 2026-07-19

### 新功能

- 新增"文件抽屉"：通过 `⌃⌥Space`、应用菜单或 Dock 右键呼出，无需切换 Finder 即可访问常用目录；内置"下载"和"桌面"标签，支持自定义文件夹。
- 文件抽屉完整功能：路径栏逐级返回、后台读取与目录监听、文件类型筛选、7 种排序、2–5 列布局、单击/多选/框选、双击打开、回车重命名、拖到其他应用、空格预览。
- 重做拖拽分屏：进入顶部热区先 Peek，继续拖入展开布局，释放前显示吸附预览。
- 使用 Icon Composer 分层图标，自动适配系统外观和旧系统。

### 体验优化

- 文件抽屉打开/关闭动画沿对应边缘滑入滑出；框选期间暂停自动隐藏。
- 设置页统一采用 macOS 系统设计语言（卡片、开关、菜单、悬停、滑杆）。
- Dock 窗口预览在 macOS 26 使用系统 Clear 液态玻璃，旧系统降级。
- 欢迎页切换缓动更从容，授权卡片使用系统液态玻璃，授权成功显示绿色勾号。
- 命中范围优化：抽屉工具栏/路径栏扩展至完整圆形；文件项目收紧至缩略图与文件名。

### 性能与稳定性

- 文件枚举、缩略图编码与缓存清理均移至后台；当前目录使用文件系统事件监听并合并短时连续变化。
- 框选自动滚动驱动原生 ScrollView，选择更新限频 60 次/秒。
- 面板显示、框选、拖拽、菜单和 Quick Look 期间暂停不必要的检测。
- SkyLight 私有符号运行时动态解析，不可用时安全降级。

### 问题修复

- 修复文件卡片抢占其他应用焦点、Peek 停住后面板无法展开、大小写不敏感磁盘无法仅改大小写等问题。
- 修复隔空投送后 `NSSharingService` 保留、空格预览桌面文件、Quick Look 提前收回、点击空白仍停留重命名等问题。
- 修复多选拖拽预览框不出现、Shift 连续选择选区丢失、多文件拖出反馈错误、框选到底部持续滚动等问题。
- 修复新截图需手动刷新、边缘 Peek 立即自动关闭、文件窗口偶发闪退式消失等问题。
- 修复快速切换目录/排序/筛选时迟到加载和选择错乱、窗口吸附后手动调整尺寸被错误恢复等问题。
- 修复快速划过 Dock 图标时过期截图覆盖当前预览、副屏位于左侧或下方时的坐标偏差。
- 修复首次启动默认设置注册过晚、辅助功能授权后监听未及时恢复、切换标签后框选覆盖层断开等问题。

### 发布说明

- 最低支持 macOS 14.0。
